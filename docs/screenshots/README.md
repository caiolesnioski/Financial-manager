# Screenshots

Esta pasta contem as screenshots do projeto para uso na documentacao.

## Como adicionar screenshots

1. Tire screenshots das principais telas do aplicativo
2. Salve as imagens nesta pasta com nomes descritivos
3. Use formato PNG ou JPG
4. Tamanho recomendado: 1280x720 pixels

## Screenshots necessarias

- [ ] `dashboard.png` - Tela principal do dashboard
- [ ] `lancamentos.png` - Lista de lancamentos
- [ ] `relatorios.png` - Pagina de relatorios com graficos
- [ ] `limites.png` - Pagina de limites de gasto
- [ ] `contas.png` - Gestao de contas (opcional)
- [ ] `login.png` - Tela de login (opcional)

## Dicas

- Capture em modo claro e escuro se possivel
- Use dados de exemplo realistas
- Evite mostrar informacoes pessoais
- Comprima as imagens para reduzir o tamanho do repositorio

## Ferramentas recomendadas

- **macOS**: Command + Shift + 4 (selecao) ou Command + Shift + 5 (opcoes)
- **Windows**: Win + Shift + S (Snipping Tool)
- **Linux**: gnome-screenshot ou flameshot
- **Compressao**: [TinyPNG](https://tinypng.com/) ou [Squoosh](https://squoosh.app/)
