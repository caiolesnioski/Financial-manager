🚀 COMO USAR
═══════════════════════════════════════════════════════════════════════════════

1. CONFIGURAR:
   • Copiar .env.example → .env
   • Preencher VITE_SUPABASE_URL e VITE_SUPABASE_ANON_KEY

2. INSTALAR:
   • npm install

3. PREPARAR BANCO:
   • Executar schema.sql no SQL Editor do Supabase

4. INICIAR:
   • npm run dev
   • Abrir http://localhost:5173

5. TESTAR:
   • Seguir instruções em GUIDE.md

6. DEPLOYAR:
   • npm run build
   • Deploy para Vercel, Netlify, ou servidor próprio
